# 01 · Спецификация Core Loop

## Источник

`GDD/3_cycle.md` — концептуальное описание трёх лупов. Здесь — конкретика для Godot-реализации.

## Микро-луп (10-30 сек)

```
Игрок ведёт аватар джойстиком к Wheat Farm → подходит → тап здания → меню
  → "Старт сбора" → NPC/игрок собирает → возвращается → пшеница в инвентаре
                                                                       ↓
                                          Игрок ведёт к Mill → тап → меню → переработка → мука
                                                                       ↓
                                          Игрок ведёт к Bakery → тап → выпечка → хлеб
                                                                       ↓
                                          Игрок ведёт к Market → тап → продажа → +Gold
```

### В Godot-сущностях

| Действие | Сущность | Сигналы |
|----------|----------|---------|
| Движение игрока | `VirtualJoystick.direction_changed` → `Player.set_movement_direction` | `Player.position_changed` |
| Тап здания | `Area3D.input_event` на `Building` (не перемещает!) | `Building.menu_opened` |
| Старт сбора | Кнопка в меню здания → `Building.start_production()` | — |
| NPC движение | `NavigationAgent3D` на `NPC` | `NPC.target_reached` |
| Сбор ресурса | `NPC.work()` / `Player.work()` → таймер | `EventBus.resource_produced` |
| Переработка | `ProductionLine.tick()` | `EventBus.resource_produced` |
| Продажа | `Building (Market).sell()` | `EventBus.resource_sold`, `currency_changed` |

### Тайминги MVP
- Wheat Farm cycle: 1.0 с
- Mill cycle: 10 с
- Bakery cycle: 30 с
- Market sell: мгновенно

## Мезо-луп (2-5 мин)

Накопление дохода → апгрейд здания / NPC / инструмента → выросло производство.

### Триггеры
- Достаточно gold → загорается кнопка `Upgrade` (визуал: glow + анимация)
- Тап → подтверждение → анимация апгрейда здания (тиры визуала)

### Game feel
- Каждый апгрейд — заметный визуал (новый меш, больше деталей)
- Звук — `upgrade_success.ogg`
- Floating text `+12% yield`
- HUD прыгает (number tween)

## Макро-луп (15-60 мин)

Открытие новой цепочки / нового здания / нового региона.

### Триггеры
- Игрок достиг level N
- Накопил достаточно gold для разблокировки
- Tutorial-флоу подводит вручную

## Раскадровка первых 30 секунд

| Время | Действие | Цель |
|-------|----------|------|
| 0 c | Splash + boot | загрузка ассетов |

## Состояния игрока

```
NEW_USER → TUTORIAL_ACTIVE → ONBOARDED → MID_GAME → PRESTIGED → LATE_GAME
```

Каждое состояние — разные UI-подсказки, разные офферы, разные ивенты.

## Friction (намеренный)

Из `GDD/3`:
- Movement speed (без апгрейда — медленно)
- Inventory capacity (нужно бежать продавать)
- Processing time (10-30 с циклы)

Friction = возможность апгрейда = эмоция «решил проблему».

В фазах:
- **Ранняя (80% active):** сильный friction, частые тапы
- **Средняя (50% active):** NPC снимают friction, появляются менеджеры
- **Поздняя (20% active):** оффлайн доминирует, активные сессии — для апгрейдов и ивентов

## Соотношение активного / пассивного

| Фаза | Активные сессии | Длительность | Главное действие |
|------|-----------------|--------------|------------------|
| Ранняя | 6-10/день | 3-5 мин | Тапы, апгрейды |
| Средняя | 4-6/день | 5-10 мин | Стратегия NPC, апгрейды |
| Поздняя | 2-3/день | 5-15 мин | Престиж, ивенты, цели |
